# Create Image Campaign

## Workflow
1. Ask for: product, audience, aspect ratio (1:1,16:9,9:16), count (1-5)
2. Generate creative direction with composition, lighting, palette
3. Create image prompts for each concept
4. Generate images
5. Return all assets with descriptions
