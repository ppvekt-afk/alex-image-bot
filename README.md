# Алекс Арт-директор — Telegram бот

Команды:
- /image текст — генерация изображения
- /ppt тема — презентация
- /brandkit название | индустрия — бренд-кит
- /generate_brand_images — генерация визуалов бренда
- /amazon название | категория | особенности — Amazon листинг
- /generate_amazon_listing — 4 изображения для Amazon
- Любой текст — генерация изображения
- презентация про тему — создание презентации
